# Vedic Kundali & Matchmaking App

A cosmic web application for generating North Indian Birth Charts and calculating Gun Milan (Matchmaking) scores.

## Features

- **Kundali Generation:** Generates a diamond-style North Indian chart based on birth details.
- **Matchmaking:** Calculates compatibility (Gunas) between two profiles.
- **Cosmic Theme:** Deep blue and gold aesthetics with twinkling star animations.
- **Responsive:** Works on Mobile and Desktop.
- **Print Friendly:** Clean print layout for reports.

## How to Run Locally

1. Open the `vedic-astrology` folder.
2. Double-click `index.html` to open it in your browser.

### Phase 1: Put Your Code on GitHub

GitHub is where your code lives online.

1. **Create a GitHub Account:** Go to [github.com](https://github.com) and sign up (it's free).
2. **Download GitHub Desktop:** Go to [desktop.github.com](https://desktop.github.com) and download/install the app.
3. **Create a New Repository:**
    - Open GitHub Desktop.
    - Click **"File"** -> **"New Repository..."**
    - **Name:** `vedic-astrology-app`
    - **Local Path:** Choose a folder on your computer where you want to save this project.
    - Click **"Create Repository"**.
4. **Copy Your Files:**
    - Open the folder you just created on your computer.
    - Copy the 3 files I gave you (`index.html`, `style.css`, `script.js`) and paste them INSIDE this new folder.
5. **Publish to GitHub:**
    - Go back to GitHub Desktop. You will see the files listed on the left.
    - In the bottom left box ("Summary"), type: `Initial commit`.
    - Click **"Commit to main"**.
    - Click the blue **"Publish repository"** button at the top right.
    - Uncheck "Keep this code private" (so it's public).
    - Click **"Publish repository"**.

### Phase 2: Host it for Free

Choose **EITHER** Render **OR** Netlify. You don't need both.

#### Option A: Publish on Netlify (Easiest)

1. Go to [netlify.com](https://www.netlify.com) and sign up.
2. Log in and click **"Add new site"** -> **"Import from existing project"**.
3. Click **"Deploy with GitHub"**.
4. Authorize Netlify to access your GitHub account.
5. Search for `vedic-astrology-app` and select it.
6. Click **"Deploy vedic-astrology-app"**.
7. Wait a few seconds. You will get a link like `random-name.netlify.app`. You can change this name in "Site Settings".

#### Option B: Publish on Render

1. Go to [render.com](https://render.com) and sign up.
2. Click **"New +"** -> **"Static Site"**.
3. Connect your GitHub account.
4. Select `vedic-astrology-app` from the list.
5. **Name:** Give it a name (e.g., `my-vedic-app`).
6. **Branch:** Leave as `main`.
7. **Publish directory:** Leave as `./` (or blank).
8. Click **"Create Static Site"**.
9. Wait a minute, and Render will give you a live URL.
